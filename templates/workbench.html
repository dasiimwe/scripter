Address & subnet
.address
Host IP with no mask
.network
Network address
.netmask
Dotted-decimal mask (e.g. 255.255.255.0)
.wildcard
Inverse mask — v4 only (e.g. 0.0.0.255)
.hostmask
Alias of .wildcard
.broadcast
Broadcast address — v4 only
.cidr
Network in addr/prefix form (e.g. 192.168.1.0/24)
.host_cidr
Host in addr/prefix form (e.g. 192.168.1.5/24)
.prefix
Prefix length as integer (e.g. 24)
Usable host range
.first
First usable host
.last
Last usable host
.hosts
Count of usable hosts (/31 → 2, /32 → 1)
.size
Total address count in the subnet
.version
4 or 6
Classification
.is_private
RFC1918 / ULA / other reserved private ranges
.is_global
Globally routable
.is_loopback
127.0.0.0/8 or ::1
.is_multicast
224.0.0.0/4 or ff00::/8
.is_link_local
169.254.0.0/16 or fe80::/10
DNS & canonical forms
.reverse_pointer
DNS PTR name (1.1.168.192.in-addr.arpa)
.exploded
Long form — useful for IPv6 logging
.compressed
Short form — useful for IPv6 config
Arithmetic

IP-aware comparison (not lexicographic)
